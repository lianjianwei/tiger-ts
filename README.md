# tiger-ts
一个使用TypeScript写的Node.js服务端基础库。

## 功能

1. API定义
    * koa
2. 配置加载器
    * yaml 文件配置加载器
3. 枚举工厂
4. 文件工厂
    * IO文件工厂
5. 数值服务
    * 默认数值处理器（累加）
    * 覆盖数值处理器
    * 过滤数值处理器
6. 数据库工厂
    * mongo 数据库工厂
7. 自定义异常类
8. 测试 mock
9. 日志模块
    * console 日志
    * log4js 日志
10. 解析器
    * number
    * string
    * bool
    * json
    * Value
    * Reward
    * Condition
    * EnumValue
11. 字符串生成器
    * Mongo ObjectId 生成器
12. 线程
13. 分布式锁
    * redis 分布式锁

## 安装

```
npm install tiger-ts
```
